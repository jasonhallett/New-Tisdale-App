CHANGELOG - Critical Cleanup Batch (1–4)
=======================================

Added
-----
- utils/body.js — shared `readRawBody(req, { maxBytes })` (10 MB default).
- utils/samsara.js — shared `samsaraGet(base, path, token, query)` with version header.

- api/auth/login.js
  • Imports `readRawBody` from utils.
  • Removed local `readRawBody` function.

- api/samsara/vehicles.js
  • Imports `samsaraGet` from utils.
  • Removed local `samsaraGet` function.

- api/samsara/vehicle-stats.js
  • Imports `samsaraGet` from utils.
  • Removed local `samsaraGet` function.

- db.js
  • `parseDataUrl` now returns `{ mime, buffer }`. (Back-compat alias: `bytes` → `buffer`.)

- auth.js
  • Safer JWT verification: length-check before `crypto.timingSafeEqual`.

Unchanged
---------
All other files are untouched.

Notes
-----
These changes are internal-only refactors/hardening; they do not alter UI/UX or API contracts.
